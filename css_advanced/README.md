# CSS Advanced Project

## Objective
This project is part of the ALX Front-End Web Development program. 
The goal is to apply CSS skills to style a webpage based on a Figma design.

## Learning Outcomes
By completing this project, I am able to:
- Understand and apply CSS selectors and specificity
- Add styles to HTML elements
- Work with Box Model properties (margin, padding, border)
- Create a fully styled web page with header, footer, banner, videos, quotes, FAQ, and membership sections
- Ensure the webpage is responsive and visually matches the Figma design.
